不可逾越的Guard2.0 之 理论篇

2024-12-06 | 不可逾越的Guard

上位如水，下位像球

被过腿应视为被降服

Guard是柔术里最重要的东西，如果你的Guard被过，不仅失去所有进攻，同时也失去所有防守，被迫进入逃命｜保命阶段，很多比赛里，过腿阶段占据了比赛大部分时间，当腿一被过，很快就被降服了，所以应该把Guard被过视为被降服一样严重的后果

主打躺姿

什么坐姿？什么时候躺姿？

坐姿

  • 在grip fighting阶段我应该坐姿
  • 如果我没有初始grip，我不应该躺下，躺下后等于放弃grip fighting

躺姿

  • grip fighting结束后，不论输赢我都应该进入躺姿
  • 对手取得grip优势，我没有了上位两个内侧把位，我需要躺下让腿参与建立框架结构
  • 我取得grip以后，我取得了上位两个内侧把位，我需要躺下建立4点控制进入Guard攻击对手

Guard的阶段

Guard三阶段

  • 接触阶段｜Engage｜抢手
  • 控制阶段｜Control｜4点控制
  • 恢复阶段｜Retention｜低于4点

在Guard阶段和控制压制一样，也是抢点游戏

  • handfighting就是手上两个内侧｜控制把位的抢夺
  • 腿部是另外两个控制把位争夺

Guard Retention

练习的是第三阶段，而且是没有手把的情况下的三阶段

  • 因为没有GRIP和Frame形成张力，所以至少需要两个Frame形成面状框架，避免单一的Frame成为杠杆
  • 如果可能，加入第3个肢体，形成立体的框架，取得真正的安全

框架和结构

Frame细分

腿

直腿｜等于送杠杆，所以直腿

Pummel是绕小腿，大腿永远Knee to Chest

  • Inside Pummel
  • Outside Pummel
  • High Pummel
  • Back Pummel
  • Shield｜小腿大腿折叠收缩
手
  • 直臂｜支撑
  • 小臂｜自我支撑
  • 大臂｜近距离支撑
  • 底手｜贴地推拉

结构

组合起来的框架，至少两个Frame参与，一主导，一辅助

组合：主导Frame + 辅助Frame

  • 组合一：Inside Pummel + Inside Pummel
  • 组合二：Outside Pummel + Outside Pummel
  • 组合三：直臂 + High Pummel
  • 组合四：底手 + Back Pummel
  • 组合五：Back Pummel + Shield + 底手

360度覆盖保护

正面｜5～7点

  • 组合一
  • 组合二

侧面｜8～10点｜2～4点

  • 组合三
  • 组合四

头顶｜11～1点

  • 组合五

常规要点

腿部位置

  • 要么双内侧
  • 要么双外侧

保护Pass Box

只留一小部分身体在地上

  • 方便旋转

不要被钉腿

不要被控头

不要被抓上半身任何地方

永远有两个Frame形成平面框架

  • 有3个形成立体框架更好，只有1个马上会变成杠杆

持续的升级你的Frame

  • 增加更多的Frame
  • 将对手推到Guard的最外层
  • Frame要控双肩
    • 单肩会被转角度破坏
  • Frame要搭配Grips使用
    • 有张力的Frame才不容易被拆掉，张力来自于压力或Grip的拉力

不做虾行

  • 会暴露Pass Box

膝盖不超出HIP

  • 会让腿变成杠杆
  • 会难以收回

不要被扭曲脊柱

提前选好一边，不要正面面对对手

薄弱点

  • 肘膝连接处
    • 塞膝盖浮固
    • 舍身斗牛
  • 南北位
    • Stack
